## Slide 3: Technical Approach

**Architecture:**
```
Arkiv (Market Data) → Hyperbridge (Cross-chain) →
ink! Contract (State Update) → Frontend (Visual)
```

**Tech Stack:**
- ink! v6 (pallet-revive) on Passet Hub → Kusama
- Arkiv Network for real-time market indexing
- Hyperbridge for trustless cross-chain messaging
- IPFS/Pinata for decentralized storage
- React + Polkadot.js frontend
